Lucid Feature #211 · task read view

Read a recorded task — its full source and result

Today a session's tasks are truncated titles in a non-interactive TaskRow — the stored sourceText + resultText are unreadable. This bundle makes the row open a read-only task detail (mirrors the Starred detail, #22) showing the full original and translated/polished text, with a per-row Open-in-workspace affordance that reuses #24's loadSource. Tapping the row reads; the trailing action loads it back into the editor — non-destructive vs. destructive, separated.

The rendering pipeline must complete every frame within a sixteen-millisecond budget. When a frame overruns, the compositor drops it and the user perceives a stutter — a brief judder rather than a full freeze.
