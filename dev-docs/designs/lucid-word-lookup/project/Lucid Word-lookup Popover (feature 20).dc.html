Lucid Feature #20 · word lookup

Word-lookup popover — click a word, read the meaning

Click any word in a rendered pane — translate result, polish original, draft, or result — and a popover opens beside it with the word, its translation, pronunciation, meaning in context, and tap-to-speak audio. This bundle specifies every state, the light/dark tokens, the phone bottom-sheet, RTL, and the focus behaviour. Headless logic (provider, prompt, segmentation, audio) is built separately.

✓Focus trap. On open, focus moves into the popover; Tab cycles play → close → play and never leaves the card until it's dismissed.
✓Keyboard dismiss. Esc closes and returns focus to the clicked word in the text, so arrow-key reading continues where it left off.
✓Labelled controls. Play is aria-label="Speak word"; while speaking it becomes “Stop”. Visible 2px accent focus ring (shown left), independent of hover.
✓Announced. role=dialog + aria-label name the word; streaming meaning is an aria-live="polite" region so the definition is read once settled, not per token.
iContrast: accent ink ≥ 4.5:1 on surface in both themes; the highlighted word keeps a 3:1 non-text indicator (the underline) so it reads without colour.
