Tighter editor resting height

The three auto-expanding editors (translate Source, polish Original, polish Draft) currently rest at an 88px inner minimum — one line of text floats over ~33px of dead space. New resting minimum: 56px inner — exactly one 18px/1.7 text line (≈31px) plus the existing 12px top + 13px bottom padding. Grow-to-content (field-sizing: content) and the ~88vh cap are unchanged.
