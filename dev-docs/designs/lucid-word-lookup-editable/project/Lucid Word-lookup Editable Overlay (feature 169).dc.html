Lucid Feature #169 · editable word lookup

Word lookup inside editable fields — read a word without leaving edit mode

The Original and Draft panes are <textarea>s — raw text with a caret, no per-word nodes — so the shipped #20 word-lookup can't reach them. This bundle specifies the missing piece: a click-segmentable overlay mirrored over the textarea, the lookup-vs-edit trigger, the word affordance, and the empty / typing / streaming behaviours. The result popover is already designed (#20) — this reuses it.
