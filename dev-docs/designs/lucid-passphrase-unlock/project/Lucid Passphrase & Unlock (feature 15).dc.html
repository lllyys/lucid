Lucid Feature #15 · WI-6

Encrypted config sync — passphrase & unlock

The one visible surface for cross-device E2E config sync. A passphrase encrypts your config and provider keys before they ever leave the device — the server only ever stores ciphertext. Crypto, codec, server and client-sync are built & headless (WI-1–5); this is the rendering layer.
