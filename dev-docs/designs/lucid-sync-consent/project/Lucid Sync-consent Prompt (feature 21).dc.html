Lucid Feature #21 · WI-3 · sync consent

First-run sync consent — “Lucid found your server. Sync, or stay local?”

A one-time, at-load prompt shown the first time Lucid detects a reachable same-origin, token-free sync server and the user hasn't yet decided. Asking once, proactively, prevents silently uploading existing local sessions (rule 65 §6). Two outcomes only: sync to my server (connect & remember) or keep local-only (stay local, never re-ask). The headless eligibility probe + consent-gated auto-connect are built separately.
