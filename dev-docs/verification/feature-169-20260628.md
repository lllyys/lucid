# Feature #169 — Word-lookup inside editable fields (mirror overlay)

Final-WI (WI-4) acceptance. All 4 WIs shipped: WI-1 owner-gated `LookupCardHost` refactor (v0.18.1), WI-2
arm-logic reducer + segments + hook (v0.18.2), WI-3 `EditableLookupOverlay` mirror component (v0.18.3), WI-4
pane wiring + ⌕ toggle + long-press (v0.19.0, PR #195, merge `c39fdf6`). Implements the committed design
`dev-docs/designs/lucid-word-lookup-editable/` (resolved #173).

- Built the production app (`pnpm build` → `dist/`, v0.19.0); served it via a token-free single-origin
