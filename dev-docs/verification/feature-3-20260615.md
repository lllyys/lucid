## Method

- `pnpm check:all` = `lint` → `typecheck` → `test:coverage` (100% on logic globs) → `build`. Green.
- Store/lib unit tests + behavioral component tests (`@testing-library/react` + `userEvent`,
  ARIA-role queries). Persistence tested via an injected backend seam (no real localStorage in the
  vitest env). No provider/live-API involvement.
- Gate-4 independent audit: 4 fresh-context subagents (Codex rate-limited — rule-48 fallback);
  log at `.claude/codex-audits/feat-feature-3-sidebar-audit.md` (follow-up-recommended; 3 High
  focus fixes + Mediums resolved; persistence/security CLEAN).

## Deferred

- **Live-browser / Playwright E2E** — Playwright is not a project dependency; behavioral ARIA tests
  cover the surfaces (consistent with features #2/#4).
- **Cross-tab live sync** — single-tab is the supported scope (documented in the plan); reload to
  see another tab's changes.
- **CJK/RTL term extraction** — extractTerms is Latin-oriented for v1 (documented limitation;
  CJK → []); a future version can add segmenter-based CJK mining.
